title: Actions overview
---
Actions are like shell commands: they take arguments and do something! TerrabaseDB currently supports the following actions: 

* [DBSIZE](dbsize)
* [DEL](del)
* [EXISTS](exists)
* [FLUSHDB](flushdb)
* [GET](get)
* [HEYA](heya)
* [MGET](mget)
* [MSET](mset)
* [MUPDATE](mupdate)
* [SDEL](sdel)
* [SET](set)
* [SSET](sset)
* [SUPDATE](supdate)
* [UPDATE](update)
